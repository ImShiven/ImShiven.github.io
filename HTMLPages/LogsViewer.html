Error Log Reader

background

It is a tedious job to browse and find most recent exception occured (among already logged hundreds of other exceptions) in error-log text file of your application.Error Log Reader parses all the exceptions from the chosen log file and represent them in a well decorated form in the order of most recent error to older ones. Thus that, errors are well readable and makes application tracking/monitoring easy as well.

One need to choose an error log file of application by hitting Read File button(shown below).As soon as file is selected, it will get parsed and errors are represented in an ordered form. Each block contains details of an error.

By default,only basic error details are shown with expand button ([+]) on top-right corner for each error block. One can click on expand button ([+]) in order to veiw full error details.

Following entities will be shown for each error block:
  • Which URL error is occured on?
  • Which Browser error is occured on?
  • Which Date & time error is occured on?
  • Which User has encountered the error?
  • What are the details of Exception?
