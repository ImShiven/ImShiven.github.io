Electronic Design

background

I have keen interest in designing and building electronic circuits & models ( I am a graduate in Electronics with major).Designed 20+ electronic models during summer internship at college. Here is a list of few microcontroller based models :
  1. Line Follower, Obstacle Detector & Remote controlled Robot
  2. Password Based Locking System
  3. RF module based remote control

1. Obstacle Dectector

8051 microcontroller based robot programmed in Embedded C which can act as Line Follower, Obstacle Dectector & can be controlled with RF module based remote in various situations. Hardware consists of RF Module, motors ,microcontroller (AT8051C), IR Sensors, multiswitch, resisters, capacitors e.t.c.

2. Password based locking system

It is password based door locking system based on 8051 microcontroller which is programmed in Embedded C. Express PCB is used to design the PCB and Proteus is used to build simulatation model.

3. REMOTE CONTROL

An RF module is used to transmit and recieve the binary data. Remote consists of RF Transmitter, an Antenna , tx chip and switches. Switches are used to transmit data. However, reciever part is placed on robot (target device). When data is being sent from Tx, Rx recieves it and device acts accordingly based on the recieved binary data.
